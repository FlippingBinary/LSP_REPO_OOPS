# LSP_REPO_OOPS

A Java console application with an assignment picker menu.

## Prerequisites

- Java 21 or later
- Gradle (or use the Gradle wrapper if available)

## Project Structure

```text
src/
├── Router.java                           # Simple CLI router
└── org/howard/edu/lsp/assignment1/
    └── HelloWorld.java                   # Assignment 1: Hello World
```

## Build and Run

Build the project:

```bash
gradle build
```

Run the application (automatically builds as well):

```bash
gradle run -q
```
